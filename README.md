## Overview

## Simple HTMX app with Node Express backend for listing and editing books + search functionality. Idea was to learn the basics of HTMX and its most common syntax, coupled with Express

## Main Features

- listing books (hx-get to backend)
- editing an already added book and updating it to backend (hx-post)
- searching the books from backend (hx-post)
- (updating the frontend with data from backend with HTMX)
